# Flaticon icon map

Code Android đang dùng asset names dưới đây. Vào Flaticon, tải SVG/PNG theo keyword tương ứng, import vào Android Studio hoặc thay file vector XML cùng tên.

| Asset file | Flaticon keyword nên tìm |
|---|---|
| ic_flaticon_lock.xml | smart lock / door lock |
| ic_flaticon_home.xml | smart home / home |
| ic_flaticon_password.xml | password / keypad |
| ic_flaticon_fingerprint.xml | fingerprint |
| ic_flaticon_face.xml | face recognition |
| ic_flaticon_card.xml | card access / access card |
| ic_flaticon_remote.xml | remote control |
| ic_flaticon_phone.xml | smartphone key |
| ic_flaticon_admin.xml | admin user / management |
| ic_flaticon_unlock.xml | unlock / remote unlock |
| ic_flaticon_settings.xml | settings |
| ic_flaticon_records.xml | history / records |
| ic_flaticon_battery.xml | battery report |
| ic_flaticon_transfer.xml | transfer / share key |
| ic_flaticon_combination.xml | combination lock |
| ic_flaticon_schedule.xml | schedule / calendar |
| ic_flaticon_nfc.xml | NFC |
| ic_flaticon_room.xml | hotel room / door room |
| ic_flaticon_alarm.xml | alarm / alert |
| ic_flaticon_gateway.xml | gateway / router / wifi |
| ic_flaticon_device.xml | devices / IoT |
| ic_flaticon_analytics.xml | analytics / report |
| ic_flaticon_hotel.xml | hotel / PMS |
| ic_flaticon_language.xml | language / translate |

Ghi chú: Flaticon free thường cần attribution; premium có thể không cần attribution. Nếu dùng bản free, thêm attribution trong màn About hoặc tài liệu sản phẩm.
